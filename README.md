# Netflix Dashboard

An interactive Tableau dashboard project built from Netflix titles data.

## Project Overview
This repository contains a Tableau workbook and supporting dataset used to explore Netflix content trends such as:
- Content type distribution
- Release year patterns
- Country-wise availability
- Genre/category insights

## Tools Used
- Tableau Desktop
- Microsoft Excel
- Data Visualization
- Data Analysis

## Files in This Repository
- **Netflix data.twb**: Tableau workbook
- **Netflix_data.xlsx**: Source dataset
- **Netflix-Dashboard.png**: Dashboard preview image

## How to Use
1. Download or clone this repository.
2. Open **Netflix data.twb** in Tableau Desktop.
3. If prompted, reconnect the data source to **Netflix_data.xlsx**.
4. Interact with filters and visuals in the dashboard.

## Preview
Open **Netflix-Dashboard.png** to view a snapshot of the dashboard.
